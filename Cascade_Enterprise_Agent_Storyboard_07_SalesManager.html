Storyboard 07 —
Sales manager · the team

The role that has been sitting unassigned in the Users table since Storyboard 02. Andrea runs the nine reps. She is the one the agent escalates to when a rep goes quiet, the one who approves the CRM cleanups, and the one who decides when the agent has judged a person unfairly.

Boards 03 to 05 all quietly hand something up to this person. The operator escalates a stock decision; the field rep who stops replying becomes a coaching item; the executive's coverage gap has to be worked by someone. The sales manager is where the agent's findings turn into conversations with people — which makes this the board where tone matters most. Every screen here is designed to help Andrea manage fairly: the agent surfaces patterns and drafts the outreach, but a human always decides what to say to another human, and can always tell the agent it got a person wrong.

Frame note

This is the most sensitive screen in the whole set, so it is the most careful. The agent describes, never diagnoses a person.

  • "Puede ser muchas cosas. No sé cuál." — the agent explicitly refuses to guess why a human behaves a certain way.
  • The suggested move is a 1:1 that leads with the person, not the metrics. "Sin cifras, primero la persona."
  • Every option is drafted; nothing sends until Andrea chooses and reviews. The agent never messages a rep about performance on its own.
  • "Solo observar otra semana" is a first-class choice. Not every dip needs an intervention.

Guardar la corrección Reconocer el apoyo a Jorge → modelo de conocimiento
Frame note

The board's most important screen. The agent judged a person on a number and got the story wrong. Andrea has the context, and the system is built to let her win.

  • Same "agent was wrong" loop as boards 03 and 04 — but here the stakes are a person's standing, so the correction has to be easy and dignified.
  • The fix is saved as a rule: cross-territory coverage now adjusts the metric for everyone, so no rep is punished for helping a colleague.
  • "Reconocer el apoyo a Jorge" turns a false flag into recognition — the opposite of what a naive dashboard would do.
  • This is the knowledge model learning fairness, which is worth more to a client than any metric.

What this board settles

The sales manager is where the agent's numbers become conversations, so this board is governed by one rule: the agent surfaces patterns and drafts words, a human decides what to say to another human. It never messages a rep about performance on its own, it leads with the person before the metric, and — most importantly — it can be told it judged someone unfairly, and it learns. Frame 05 is the whole thesis of the platform aimed at people instead of data: a human always has the last word, and being corrected is how the system earns the right to be trusted with a team.

Components this adds

  • Coaching item with drafted-not-sent outreach options
  • Manager-seat approval gate (same component)
  • Collections watch + reminder run
